1. CSS IF() Price Colors

@function --price-color() {
    result: if(
        style(--product-state: sale): #e11d48;
        style(--product-state: clearance): #dc2626;
        style(--product-state: premium): #059669;
        else: #374151
    );
}

2. CSS IF() Theme Colors

@function --theme-bg() {
    result: if(
        style(--theme-mode: dark): #1f2937;
        style(--theme-mode: blue): #1e40af;
        style(--theme-mode: green): #166534;
        else: #ffffff
    );
}

Theme-Aware Content

Background, text color, and border style all change via CSS IF() functions.

3. CSS IF() Notification System

@function --notification-bg() {
    result: if(
        style(--alert-type: error): #fecaca;
        style(--alert-type: warning): #fef3c7;
        style(--alert-type: success): #d1fae5;
        style(--alert-type: info): #dbeafe;
        else: #f3f4f6
    );
}
System Notification
This notification changes its entire appearance (colors, icons, borders) based purely on CSS IF() functions responding to the --alert-type custom property.
